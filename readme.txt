Free future value calculator that uses the formula FV = PV (1 + I/Y)n to find the future value of an investment with periodic deposits.

[https://www.calculator.io/future-value-calculator/](https://www.calculator.io/future-value-calculator/)

== Usage ==

Add the Future Value Calculator shortcode to your page, post or sidebar:

`[ci_future_value_calculator]`

Add the following code to your template where you would like the Future Value Calculator to appear:

`<?php display_ci_future_value_calculator(); ?>`

== Installation ==

1. Upload the Future Value Calculator /ci_future_value_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Future Value Calculator plugin through the "Plugins" menu in WordPress.
